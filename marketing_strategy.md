# KiQ Robotics 柔軟指 マーケティング戦略

**作成日**: 2026-03-26
**目標**: 6ヶ月で受注件数10倍（月6-8件 → 月60-80件）、商社依存からの脱却
**制約**: 4名体制、週10時間、月額10万円以下

---

## 戦略の全体像

### 現状の構造的問題

```
現在のファネル:
  認知 → 商社が提案 → 伝言ゲーム → ミスマッチ → 「高い」で失注
                                                    ↑ ここが詰まっている

目指すファネル:
  認知 → 顧客が自分で見つける → 課題がマッチ → 即決（1ヶ月以内）
         ↑ ここを作る
```

データが示す事実:
- 直販は件数14%だが売上32%。単価は商社経由の**2.8倍**。リードタイム**1ヶ月 vs 半年**
- 正しい課題にマッチすれば価格は障壁にならない（失注の実態は「入口の間違い」）
- 製造業の製品認知の**40%がGoogle検索経由**（メディックス2024年調査）
- 営業経由の認知は年々低下。Webでの自発的な情報収集にシフトしている

→ **商社を置き換えるのではなく、「顧客が自分でKiQを見つけて問い合わせてくる」仕組みを作る**

### 戦略の3本柱

| 柱 | 目的 | 期待効果 |
|:---|:---|:---|
| **1. 検索で見つかる** | バーニングニーズを持つ担当者がGoogle検索したとき、KiQが出てくる状態を作る | 月間問い合わせ数の大幅増 |
| **2. 動画で信頼を得る** | 「本当に掴めるのか？」を動画で即座に証明する | 問い合わせ→受注の転換率向上 |
| **3. 仕組みで回す** | リード管理・フォローアップを自動化し、4名で60件/月を捌ける体制を作る | 工数を増やさずスケール |

## 柱1: 検索で見つかる（SEO + ポータル + 広告）

### なぜ検索が最優先か

- 製造業の認知チャネル1位: Google検索（40.2%）
- 「柔軟指」「ソフトグリッパー」関連の日本語コンテンツは極めて少ない → SEOで上位表示の余地が大きい
- McHone Industries（米国の家族経営金属加工メーカー）が6ヶ月でリード1,500%増を達成した事例も、「製造業はオンラインコンテンツが少ないため、高ボリューム・低競争キーワードで容易に上位表示できた」ことが成功要因

### 施策1-1: イプロス無料出展（即日・0円）

| 項目 | 内容 |
|:---|:---|
| **何をするか** | イプロスものづくりに柔軟指の製品ページを作成し、カタログPDFをアップロードする |
| **なぜ効くか** | 閲覧会員180万人、年間引き合い223万件。製造業の購買担当者が日常的に使うプラットフォーム。商談化率10〜15%の実績 |
| **費用** | 0円（無料プラン） |
| **工数** | 初期3〜5時間、以降月1時間 |
| **期待効果** | 月5〜15件の問い合わせ獲得 |

**ポイント**: 製品説明に「把持ミス削減」「多品種対応」「段取り替え不要」等の**課題ワード**を入れる。「柔軟指」という製品名ではなく、**顧客の課題**で見つかるようにする

### 施策1-2: メトリー掲載（即日・成果報酬型）

| 項目 | 内容 |
|:---|:---|
| **何をするか** | メトリーの「ロボットハンド」「グリッパー」カテゴリに掲載する |
| **なぜ効くか** | 購入意向が高いユーザーが集まるため、問い合わせが具体的で受注に繋がりやすい |
| **費用** | 成果報酬型（少額） |
| **工数** | 初期2〜3時間 |

### 施策1-3: 自社サイトSEO（0円・工数のみ）

**ターゲットキーワード（課題ベース）**:

| 検索意図 | キーワード例 | 対応コンテンツ |
|:---|:---|:---|
| 把持ミスで困っている | 「ロボット 把持ミス 対策」「グリッパー つかみ損ね」 | 事例記事: 「把持ミスによるライン停止を解消した方法」 |
| 多品種対応したい | 「1つのハンドで 複数ワーク」「段取り替え 削減」 | 事例記事: 「1つの柔軟指で5種類のワークに対応」 |
| ワーク損傷を防ぎたい | 「ワーク 傷つけない ロボットハンド」「ソフトグリッパー」 | 技術解説: 「ラティス構造が対象物を傷つけない理由」 |
| 比較検討中 | 「OnRobot 代替」「Robotiq 互換 グリッパー」 | 比較記事: 「柔軟指 vs メカチャック 比較ガイド」 |

月2〜3本の記事を自社サイトに掲載する。1本あたり2〜3時間。

### 施策1-4: Google広告（月3〜5万円）

| 項目 | 内容 |
|:---|:---|
| **何をするか** | 上記キーワードでGoogle検索広告を出稿する。**愛知県に地域限定** |
| **なぜ効くか** | SEOは効果が出るまで3〜6ヶ月かかる。広告は即日で検索上位に表示できる。地域を絞れば少額でも効率的 |
| **費用** | 月3〜5万円 |
| **工数** | 初期3時間、以降月2時間 |
| **期待効果** | 月10〜30件のサイト訪問、うち5〜15%が問い合わせ |

## 柱2: 動画で信頼を得る（SNSコンテンツ戦略）

### 現状の評価

松田さんの運用は「平日毎日1本、3ヶ月で全プラットフォーム合計300本弱」と、ミスミmeviyの「3ヶ月86本」を上回るペース。**量は十分。質の最適化で大きく伸びる余地がある**。

参考事例: ミスミmeviy
- 2名体制で3ヶ月86本 → 接触人数12.2倍、初回購入5倍、ROAS目標の20倍
- 成功要因: 顧客目線の徹底、教育+エンターテインメントの組み合わせ

### プラットフォーム優先順位の見直し

| 優先度 | プラットフォーム | 理由 | 方針 |
|:---:|:---|:---|:---|
| **1** | **YouTube** | 検索による長期発見性がBtoBに最適。エンゲージメント率5.91%で最高。既に直近平均2,813再生 | 主戦場。ここに集中投資する |
| **2** | **Instagram Reels** | フォロワー24人で平均1,559再生。リーチ効率が全プラットフォーム最高 | YouTubeの横展開で低工数維持 |
| **3** | **TikTok** | フォロワー211、平均914再生。バズ時のインパクト大（トヨタ事例8万再生） | YouTubeの横展開で低工数維持 |
| **4** | **Facebook** | フォロワー228だがリーチ効率低い | 横展開のみ。追加工数かけない |
| **撤退検討** | **X** | フォロワー9、平均80再生。投資対効果なし | 横展開の自動投稿のみ。手動運用は停止する |

### コンテンツミックスの設計

データから明確にわかっている「刺さるもの」と「刺さらないもの」:

**刺さる（再現する）**:
1. 身近なもの×柔軟指の意外性（ポテチ12,900再生、スライム4,445再生）
2. Normal Hand vs. Lattice Hand 比較対決（水2,911再生、から揚げ2,052再生）
3. 有名企業の導入事例（トヨタ3,384再生、TikTok 8万再生）

**刺さらない（やめる）**:
1. 自社宣伝色が強いもの（認定式719再生、開発事例88再生）
2. 無機質な工業部品だけの把持試験（387〜837再生）
3. 知名度の低い企業名を出す事例（リックス260再生）

### 週5本のコンテンツ配分

| 曜日 | カテゴリ | 内容例 | 狙い |
|:---|:---|:---|:---|
| 月 | **バズ狙い**（身近なもの） | 豆腐、生卵、シュークリーム等を掴む | 認知拡大。フォロワー獲得 |
| 火 | **比較対決** | Normal Hand vs. Lattice Hand | 製品優位性の可視化 |
| 水 | **課題解決**（産業用途） | 「把持ミス率○%削減」「段取り替えゼロ」 | リード獲得。検索SEO |
| 木 | **バズ狙い**（意外な使い方） | UFOキャッチャー、書道、料理等 | 認知拡大 |
| 金 | **導入事例** | トヨタ・デンソー・ボッシュの事例（企業名を前面に） | 信頼構築。BtoB意思決定者向け |

### 制作効率化

- **バッチ撮影**: 月2回の撮影日で2週間分を撮り溜める（1回の撮影で5〜8素材）
- **リパーポース**: 1本の素材からYouTube Shorts → Instagram Reels → TikTok → Facebookに展開。プラットフォーム別の再編集は最小限にする
- **推奨尺**: 15〜35秒。最初の2〜3秒にフック（最も衝撃的なシーンを冒頭に持ってくる）
- **タイトルSEO**: YouTube動画のタイトル・説明文に課題キーワードを入れる（例: 「【把持ミス対策】柔軟指でポテチを掴んでみた」）

## 柱3: 仕組みで回す（CRM + 自動化）

### なぜ仕組みが必要か

月60件の受注を4名で回すには、手動での管理は不可能。現在はCRM・MAツールがゼロの状態。

参考事例:
- コーセル: MA導入で問い合わせ10倍、新規顧客売上前年比30%増
- キトー: Web経由インサイドセールスで初年度目標500%超達成
- Advanced Piping Products: HubSpot導入、専任マーケターなしでオーガニックトラフィック408%増

### 推奨ツールスタック（月額約2,000円〜）

| カテゴリ | ツール | 月額 | 用途 |
|:---|:---|---:|:---|
| **CRM** | HubSpot CRM（無料プラン） | 0円 | リード管理、商談パイプライン、タスク管理 |
| **MA** | BowNow（無料プラン） | 0円 | Webサイト訪問企業の特定、リードスコアリング |
| **LP** | ペライチ（ライトプラン） | 1,456円 | 課題別ランディングページの作成 |
| **SEO** | Google Search Console | 0円 | 検索パフォーマンスの測定 |
| **SNS分析** | SocialDog（無料プラン） | 0円 | 投稿分析（将来的に必要になれば） |
| **合計** | | **約1,500円** | |

### リード管理フロー

```
[問い合わせ発生]
    ↓
[HubSpotに自動登録]
    ↓
[課題ヒアリング（メール or 電話）]
    ↓ バーニングニーズあり？
    ├─ YES → 把持テスト提案 → 受注（1ヶ月以内）
    └─ NO → ナーチャリング（メール定期配信）→ 将来の案件化
```

## リックスとの棲み分け

リックスを切るのではなく、**バイパスする**:

| チャネル | 対象 | 役割 |
|:---|:---|:---|
| **リックス経由** | 愛知県の自動車関連大手（現状維持） | 既存の関係を活かす。改善は求めない |
| **直販（Web経由）** | リックスがカバーしていないエリア・セグメント | KiQが自分でリードを獲得・クロージングする |
| **他商社** | 日伝・マルエム商会・豊田油気等 | 実績のある商社は継続。ただし直販の優先度を上げる |

キトーの事例に学ぶと:
- 「代理店がカバーしていない領域を直販で獲得する」棲み分けが最も現実的
- Dow Corningのように直販チャネルを明確に分離すれば、チャネルコンフリクトを回避できる
- リックスに対しては「Web経由で来た案件はKiQが直接対応する」と明確にする

## 10倍は可能か？ — 正直な評価

**3ヶ月で3倍（月20件）は高い確実度で達成可能**:
- イプロス出展だけで月5〜15件の問い合わせ増が見込める
- Google広告で月5〜10件のサイト訪問→問い合わせ
- 既存リピート顧客の横展開で月3〜5件

**6ヶ月で10倍（月60件）はストレッチだが、複数のチャネルが同時に立ち上がれば射程圏内**:
- イプロス（有料含む）: 月20〜30件
- Google広告+SEO: 月15〜25件
- SNS経由: 月5〜10件
- 既存顧客リファラル: 月5〜10件
- 展示会（出展した場合）: 月10〜20件（展示会後のフォローアップ含む）

**最大のリスク**: 問い合わせは増えても、4名で対応できる商談数に物理的な上限がある。CRMとテンプレート化による効率化が不可欠。
