# 自動車関連メーカー 生産技術部・製造部へのリーチ戦略調査

**作成日**: 2026-03-26
**対象**: 日本の自動車関連メーカーの生産技術部・製造部（担当者〜管理者）
**前提**: 月額予算10万円以下、4名体制（北九州拠点）、愛知県・東海エリア重点

---

## 1. 製造業エンジニアの情報収集チャネル（調査データに基づく）

### メディックス社調査（2024年版、製造業製品選定者対象）

製造業の製品選定に関与する担当者が「認知のきっかけ」として使う情報源の割合（2024年）:

| 順位 | 情報収集チャネル | 割合 | 備考 |
|:---:|:---|:---:|:---|
| 1 | **検索エンジン（Google等）** | **40.2%** | 3年連続で最も高い。2019年・2021年・2024年で一貫して1位 |
| 2 | 展示会 | 上位 | コロナ後に回復傾向 |
| 3 | メーカー・ベンダーのWebサイト | 上位 | 検索エンジン経由での流入が多い |
| 4 | カタログ・技術資料 | 中位 | 紙・PDF両方 |
| 5 | 出入り業者・営業担当 | 低下傾向 | 2024年が最も低い（コロナ前から減少が継続） |
| 6 | SNS | **9.9%** | 2019年→2024年で上昇傾向。約1割が認知のきっかけ |

**出典**: 株式会社メディックス「BtoBマーケティングアンケート調査結果【2024年版】」（2019年/2021年/2024年の3回比較調査）

### 主要な傾向

- **検索エンジンが圧倒的に重要**: 製造業の担当者は課題が発生した時点でまずGoogleで検索する。SEO対策の優先度は極めて高い
- **営業経由の認知は縮小**: 出入り業者やメーカー営業からの情報提供は年々低下しており、Webでの自発的な情報収集にシフトしている
- **SNSは成長中だがまだ1割**: 認知チャネルとしてのSNSは上昇傾向にあるが、直接的なリード獲得より「認知・ブランディング」としての位置づけ

### 3-4. 展示会の費用対効果に関する調査データ

アペルザ社の展示会来場者調査（413名回答）による知見:

- **来場目的**: 「市場トレンド調査・定期的な情報収集」が最多。一方で80%超が「出展企業からの詳細説明を聞く」「特定の企業・製品を見る」とも回答。既に認知のある企業・製品の詳細確認の場でもある
- **ブース配布物**: 94.9%が全ての資料は持ち帰らない（67.6%が必要なもののみ選択）
- **展示会後の行動**: 83.4%が社内向けの展示会レポートを作成。レポートには競合との差異や導入効果など、カタログにない情報が記載される
- **導入検討**: 56.6%が展示会訪問後に導入検討を開始

**展示会で成果を出すためのポイント**:
- 事前にWeb・SNSで認知を獲得しておき、「あの会社のブースに行こう」と思わせる
- ブース配布物は「社内レポートに書きやすい」情報（競合比較、導入効果の数値）を含める
- 展示会後のフォローアップ（メール等）が成約率を大きく左右する

## 4. LinkedIn日本の製造業利用実態

| 項目 | 内容 |
|:---|:---|
| **日本のユーザー数** | 約400万人（2025年時点）。日本の人口比3%未満 |
| **グローバル** | 12億人超 |
| **日本での位置づけ** | 「最大の未開拓市場」と呼ばれる。IT・デジタル分野の利用が先行 |
| **製造業での利用** | 限定的。日本の製造業エンジニアの多くはLinkedInを日常的に使っていない |
| **利用が多い層** | 外資系企業勤務者、海外事業関連、転職検討者 |
| **成長傾向** | 2025年Q2に急増。徐々に浸透しつつある |

### KiQにとっての評価

- **短期的な優先度: 低い**: 日本の自動車関連メーカーの生産技術部担当者がLinkedInで情報収集している割合は極めて低い。月10万円の予算をLinkedIn広告に投じる費用対効果は見込みにくい
- **例外**: トヨタ・デンソー・ボッシュ等の大手企業の中でも海外拠点との連携が多い部門はLinkedIn利用率が相対的に高い可能性がある
- **中長期的には検討余地あり**: 日本でのLinkedIn浸透が進めば、生産技術部の管理者層へのリーチ手段として有効になる可能性がある

## 5. 自動車関連メーカー生産技術部の新規ツール・部品発見プロセス

### 典型的な発見〜導入フロー

```
[課題発生] → [情報収集] → [候補選定] → [テスト・検証] → [社内稟議] → [導入]
```

### 各フェーズの詳細

#### フェーズ1: 課題発生（トリガー）
- 把持ミスによるライン停止が発生する
- ワーク変更に伴いチャック交換が必要になる
- 既存の爪・指が摩耗して品質が低下する
- 多品種対応のためにチャックの段取り替え工数が増加する

#### フェーズ2: 情報収集（認知のきっかけ）
| 手段 | 重要度 | 備考 |
|:---|:---:|:---|
| **Google検索** | 最重要 | 「ロボットハンド 把持ミス」「グリッパー 多品種対応」「柔軟グリッパー」等で検索。認知の40%がここから |
| **イプロス等のポータル** | 高 | 製品カテゴリから比較検討 |
| **展示会** | 高 | 年1〜2回の定期情報収集。既知製品の詳細確認も兼ねる |
| **商社・代理店からの提案** | 中（低下傾向） | 出入りの商社が持ち込む。ただしこのチャネルは年々縮小 |
| **同僚・社内の口コミ** | 中 | 他拠点での導入事例の社内共有 |
| **専門メディア記事** | 中 | MONOist、日経クロステック等の技術記事 |
| **SNS（YouTube等）** | 低〜中（上昇中） | 認知のきっかけとして約10%。動画による把持デモは効果的 |

#### フェーズ3: 候補選定
- 複数メーカーのカタログ・技術資料を比較する
- 価格・納期・互換性（OnRobot/Robotiq互換等）を確認する
- サンプル送付やデモを依頼する

#### フェーズ4: テスト・検証
- 実際のワークで把持テストを実施する
- サイクルタイム・把持成功率・ワーク損傷の有無を評価する

#### フェーズ5: 社内稟議
- 単価20万円前後であれば現場担当者レベルで決裁できることが多い（KiQの実績と一致）
- 社内展示会レポートが稟議資料として活用されることもある

## 8. 「バーニングニーズ」を持つ担当者の探し方

### バーニングニーズの定義（KiQの文脈）

- 把持ミスによるライン停止が繰り返し発生している
- ワーク損傷・廃棄による損失が発生している
- 多品種対応のための段取り替えに過大な工数がかかっている
- 既存の爪・指の摩耗が早く、交換頻度が高い

### 探し方: 具体的な手法

#### 手法1: 検索キーワードからの逆引き（最優先）

バーニングニーズを持つ担当者は「課題ベース」で検索する。以下のキーワードで自社サイト・LP・イプロスページがヒットするようにする:

| 検索意図 | 想定キーワード例 |
|:---|:---|
| 把持ミスで困っている | 「ロボット 把持ミス 対策」「グリッパー つかみ損ね」「チャック 滑り 改善」 |
| 多品種対応したい | 「1つのハンドで 複数ワーク」「段取り替え 削減 グリッパー」「汎用グリッパー」 |
| ワーク損傷を防ぎたい | 「ワーク 傷つけない ロボットハンド」「ソフトグリッパー」「柔軟グリッパー」 |
| 爪・指の摩耗が早い | 「ロボットハンド 爪 摩耗」「グリッパー 耐久性」「フィンガー 交換頻度」 |

**アクション**: これらのキーワードに対応する課題解決型のコンテンツ（ブログ記事・技術資料・動画）を作成し、自社サイト・イプロス・YouTubeに掲載する

#### 手法2: イプロスでのカテゴリ露出

イプロスでは「ロボットハンド」「グリッパー」「エンドエフェクタ」等のカテゴリページに製品が表示される。バーニングニーズを持つ担当者はこのカテゴリで製品を比較検討するため、製品説明に「把持ミス削減」「多品種対応」等の課題ワードを含めて掲載する

#### 手法3: 展示会での課題ヒアリング

展示会では「こういう課題ありませんか？」と問いかけるブースメッセージが有効。具体的には:
- 「把持ミスでライン停止していませんか？」
- 「チャック交換の段取り替え、月に何回ですか？」

このような質問型のキャッチコピーをブースに掲示し、バーニングニーズを持つ来場者を引き寄せる

#### 手法4: 既存顧客からの紹介

最もバーニングニーズに近い見込み客は、既存顧客の社内にいる。トヨタの導入実績を持つ担当者に「同じ課題を抱えている他部署や協力会社はありませんか？」と聞く

#### 手法5: YouTube動画でのニーズ喚起

既に反応が良い「vs比較」形式（Normal Hand vs. Lattice Hand）は、バーニングニーズの可視化に有効。「メカチャック vs 柔軟指: 把持ミス率の比較」のような動画は、課題を抱える担当者が検索で見つけやすい
