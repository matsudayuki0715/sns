# SNS運用改善ガイド（松田さん向け）

**目的**: 現在のSNS運用の「量」を維持しつつ、「質」を最適化して問い合わせに繋げる
**前提**: 1日2時間、1本あたり1時間、平日毎日1本の投稿ペースは変えない

---

## 1. 変更点サマリー

| 項目 | 現状 | 変更後 |
|:---|:---|:---|
| **主戦場** | 5プラットフォーム均等 | YouTube最優先。他は横展開 |
| **コンテンツ** | 毎日その場で企画 | 週5本の曜日別テーマ固定 |
| **撮影** | 毎日撮影+編集 | 月2回のバッチ撮影（1回で5〜8素材） |
| **投稿先** | 5つ個別に投稿 | YouTube→他4つに横展開 |
| **X** | 手動投稿 | 自動投稿のみ。手動運用停止 |
| **タイトル** | 自由 | SEOキーワードを含める |

---

## 2. プラットフォーム別の運用方針

### YouTube（最優先 — ここに集中する）

- **投稿形式**: Shorts（縦型、60秒以内）
- **なぜ最優先か**: 検索による長期発見性がBtoBに最適。エンゲージメント率5.91%で全プラットフォーム最高。製造業の担当者がGoogle/YouTube検索で把持課題を調べる
- **現状**: 登録151、直近平均2,813再生、68本 → 全プラットフォームで最も伸びている

**YouTubeだけ追加でやること**:
- タイトル・説明文にSEOキーワードを入れる（後述）
- ハッシュタグを統一する
- 既存68本のタイトル・説明文も順次改修する（Phase 2で）

### Instagram Reels（第2優先 — 横展開）

- フォロワー24人で平均1,559再生。リーチ効率が異常に高い
- YouTubeで作った動画をそのまま投稿するだけでOK
- 追加編集は不要

### TikTok（第3優先 — 横展開）

- フォロワー211、平均914再生
- YouTubeで作った動画をそのまま投稿
- TikTokはバズの爆発力がある（トヨタ事例8万再生）。バズ狙いコンテンツとの相性が良い

### Facebook（第4優先 — 横展開のみ）

- フォロワー228だがリーチ効率が低い
- 横展開の自動投稿のみ。追加工数はかけない

### X（運用停止）

- フォロワー9、平均80再生。投資対効果なし
- 自動投稿ツール（後述）で横展開するだけ。手動での投稿・返信は停止する
- 停止で浮いた時間をYouTubeのタイトル最適化に使う

## 3. 曜日別コンテンツカレンダー

### 基本ルール

- **月〜金の5本**をテーマ固定にする
- テーマを固定することで「今日何撮ろう」の迷いが消える
- 視聴者にも「火曜は比較対決の日」とリズムが伝わる

### カレンダー

| 曜日 | テーマ | 内容 | 狙い |
|:---|:---|:---|:---|
| **月** | バズ狙い（身近なもの） | 日常の食べ物・雑貨を柔軟指で掴む | フォロワー獲得・認知拡大 |
| **火** | 比較対決 | Normal Hand vs. Lattice Hand | 製品優位性を視覚で証明 |
| **水** | 課題解決（産業用途） | 実際の産業ワークの把持デモ | 問い合わせ獲得・SEO |
| **木** | バズ狙い（意外な使い方） | エンタメ要素のある状況設定 | フォロワー獲得・認知拡大 |
| **金** | 導入事例 | 有名企業での実際の活用 | BtoB信頼構築 |

### 各テーマの詳細と企画例

**コンセプト**: 「え、それ掴めるの？」という驚きを与える

**企画ストック（毎週1つ使う）**:

| # | 対象物 | 見どころ |
|:---:|:---|:---|
| 1 | 生卵 | 割れずに掴めるか？の緊張感 |
| 2 | 豆腐（絹ごし） | 究極の柔らかさに対応 |
| 3 | シュークリーム | 潰さずに掴む |
| 4 | 風船 | 割れるか割れないか |
| 5 | 氷 | 滑る+溶ける二重の難しさ |
| 6 | こんにゃく | ぬるぬる×柔らかい |
| 7 | マシュマロ | ふわふわを潰さない |
| 8 | 寿司 | 崩さずに持ち上げる |
| 9 | ぶどう（1粒） | 小さい+潰れやすい |
| 10 | たこ焼き | 熱々を崩さず |
| 11 | プリン | 皿から持ち上げる衝撃 |
| 12 | 紙コップ（水入り） | 潰さず倒さず |

**撮影のポイント**:
- 最初の2秒で「何を掴むか」を見せる（フック）
- 掴む瞬間はスローモーションで
- BGMはポップなもの（著作権フリー）
- 尺: 15〜30秒

#### 火曜: 比較対決（Normal Hand vs. Lattice Hand）

**コンセプト**: 普通のグリッパーと柔軟指を並べて、同じワークを掴わせる

**これまでの実績**:
- 水（グラス）: 2,911再生
- から揚げ: 2,052再生

**企画ストック**:

| # | ワーク | 見せたいポイント |
|:---:|:---|:---|
| 1 | ペットボトル（空 vs 満杯） | 形状変化への対応力 |
| 2 | りんご vs みかん | サイズ違いを1つの指で |
| 3 | ゴムボール | 柔軟物の把持安定性 |
| 4 | 紙パック | 潰さずに掴めるか |
| 5 | 缶（丸）vs 箱（四角） | 形状の違いを1つの指で |
| 6 | 陶器の器 | 傷つけずに掴む |
| 7 | 工具（レンチ等） | 不定形の金属部品 |
| 8 | 布（タオル） | 柔軟物の難しさ |

**撮影のポイント**:
- 画面を左右分割。左: Normal Hand、右: Lattice Hand
- Normal Handが失敗→Lattice Handが成功、の順番
- テロップ: 「Normal Hand: FAIL」「Lattice Hand: SUCCESS」
- 尺: 20〜35秒

**なぜ重要か**: これが直接的に問い合わせに繋がるコンテンツ。YouTube検索でこの動画が見つかることが目標

**企画ストック**:

| # | ワーク | 訴求する課題 |

| 1 | 自動車部品（複数種類を1つの指で） | 多品種対応・段取り替え削減 |
| 2 | コネクタ | 小型精密部品の把持 |
| 3 | ボトル（形状違い複数） | 多品種対応 |
| 4 | パイプ・ホース | 柔軟物の安定把持 |
| 5 | 鋳物部品 | 形状が不均一な重量物 |
| 6 | 樹脂成形品 | 傷つけやすいワークの保護 |
| 7 | 食品容器 | 食品業界への訴求 |
| 8 | アルミ部品 | 表面傷防止 |

**撮影のポイント**:
- 冒頭テロップ: 「把持ミスでライン停止していませんか？」等の課題提起
- 実際にワークを掴む→離す→複数回繰り返して安定性を見せる
- 終了テロップ: 「無料把持テスト受付中 → kiq-robotics.co.jp」

#### 金曜: 導入事例

**コンセプト**: 「あのトヨタも使っている」という信頼性

**これまでの実績**:
- トヨタ自動車: YouTube 3,384再生、TikTok 8万再生 ← 圧倒的

**掲載可能な範囲で**:

| # | 内容 | テロップ例 |
|:---:|:---|:---|
| 1 | トヨタでの実際の把持シーン | 「トヨタ自動車で採用。把持ミスゼロ化」 |
| 2 | デンソーでの運用風景 | 「デンソーで安定稼働中。リピート10件」 |
| 3 | 万博出展の様子 | 「大阪・関西万博で展示」 |
| 4 | ボッシュでの高精度把持 | 「ボッシュのインジェクタラインで採用」 |
| 5 | 導入効果の数字まとめ | 「130件超の導入実績。リピート率33%」 |

**掲載NGの場合の代替**:
- 「大手自動車メーカーで採用」

- なければ、社内ラボで同じワークを再現して撮影
- 信頼性を出すテロップ: 企業ロゴ、件数、リピート率
- 尺: 30〜45秒

## 4. 制作フローの効率化

### バッチ撮影（月2回）

**現状の問題**: 毎日撮影+編集で1時間。企画に悩む時間もあり、クオリティにムラが出る

**改善**: 月2回の撮影日を設定し、1回で5〜8素材を撮り溜める

#### 撮影日のスケジュール例（3時間ブロック）

```
09:00-09:30  準備（機材セット、ワーク用意）
09:30-10:00  月曜素材 × 2本（身近なもの）
10:00-10:30  火曜素材 × 2本（比較対決）
10:30-11:00  水曜素材 × 2本（産業用途）
11:00-11:30  木曜素材 × 2本（意外な使い方）
11:30-12:00  金曜素材 × 1〜2本（導入事例）
```

→ 3時間で約2週間分（8〜10本）の素材が撮れる

#### 毎日の作業（1時間以内）

```
[編集 30〜40分]
  ↓
[YouTube投稿 + タイトル・説明文・ハッシュタグ設定 10分]
  ↓
[他プラットフォームに横展開 10分]
```

### リパーポース（横展開）の手順

1本の動画を5プラットフォームに投稿する手順:

```
Step 1: YouTubeにShortsとして投稿（タイトル・説明文をSEO最適化）
Step 2: 同じ動画ファイルをInstagram Reelsに投稿
Step 3: 同じ動画ファイルをTikTokに投稿
Step 4: 同じ動画ファイルをFacebookに投稿
Step 5: Xに自動投稿（後述のツールで設定）
```

**プラットフォーム別の微調整（最小限）**:
- **YouTube**: タイトルにSEOキーワード。説明文に自社サイトURL
- **Instagram**: ハッシュタグを10〜15個（後述のリスト参照）
- **TikTok**: トレンドの音源があれば差し替え（なければそのまま）
- **Facebook/X**: そのまま

## 6. 既存動画68本の改修（Phase 2）

現在の68本のYouTube動画のタイトル・説明文をSEO最適化する。
一気にやらなくてOK。1日3〜5本ずつ、2〜3週間で完了する。

### 改修の優先順位

1. **再生数が高い動画から**（検索で見つかる可能性が高い）
2. **課題解決・導入事例の動画**（問い合わせに直結する）
3. **残りのバズ系動画**

### 改修内容

- タイトルにSEOキーワードを追加する
- 説明文をテンプレートに統一する（自社サイトURL、製品特長、ハッシュタグ）
- サムネイルの統一（余裕があれば）

## 7. やめること（明確に）

| やめること | 理由 |
|:---|:---|
| Xの手動投稿・返信 | フォロワー9、平均80再生。時間の無駄 |
| 自社宣伝色の強いコンテンツ | 認定式88再生、開発事例88再生。誰も見ない |
| 無名企業の導入事例 | リックス260再生。知名度がないと刺さらない |
| 無機質な工業部品だけの把持試験 | 387〜837再生。面白みがなく見てもらえない |
| 完璧を目指した長時間編集 | 80%の完成度で72時間以内に出す方が効果的 |

再生数が平均の2倍以上になった動画は、以下を記録する:

- 冒頭のフックは何だったか

→ パターンが見えてきたら、そのパターンの頻度を上げる
